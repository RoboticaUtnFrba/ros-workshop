# ros-jupyter-docker

Dockerfile for Robot Operating System and Jupyter Notebooks

## Before...

```sh
./build.sh
```

## Try it!

```sh
./run.sh
```

Open a web browser and type localhost:8888. Insert the **password**.

## After

```sh
./stop.sh
```
